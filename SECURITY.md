# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 0.3.x   | :white_check_mark: |
| < 0.3   | :x:                |

## Reporting a Vulnerability

This project is a terminal tool that does not handle network requests beyond
fetching lyrics from public APIs (LRCLIB). It does not process user uploads,
store credentials, or run anywhere but the user's local machine.

If you find a security issue:

1. **Do not** open a public GitHub issue.
2. Email the maintainer directly (see GitHub profile) or report via
   [GitHub Security Advisories](https://github.com/RicknotDev/lrc-tools/security/advisories).

You should expect a response within 7 days. If the issue is confirmed, a fix
will be shipped in the next patch release.
